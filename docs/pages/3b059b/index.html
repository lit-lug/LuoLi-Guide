    # 教务处官网

    • 点此进入教务处 (opens new window)，或复制此链接 http://www.lit.edu.cn/jwc/ 至浏览器打开。

    # 洛理教务系统登录入口

    • 点此进入教务系统登录入口 (opens new window)，或复制此链接 https://www.lit.edu.cn/jwc/jwwlglxt.htm 至浏览器打开。

    # 教务系统登录说明

    提示

    此页面有五个入口，其中入口一四五为校内网访问，其余入口二三为外网访问。
    校内网访问进入教务系统首页登录即可。
    外网访问需要先登录智慧门户，然后从智慧门户进入教务系统。

    # 注意

    • 教务系统密码与智慧门户系统密码并不相同。
    • 教务密码忘记请联系各自学院管理人员重置密码。

    # 成绩查询

    # 登录教务管理系统(https://www.lit.edu.cn/jwc/jwwlglxt.htm)，单击【考试安排】--【考试安排表】可查看考试的时间和地点。

    # 登录教务系统(https://www.lit.edu.cn/jwc/jwwlglxt.htm)，单击【学生成绩】--【查看成绩】--【检索】，可查询成绩。

    # 重修报名及缴费

    # 1.登录教务在线—教务管理系统(https://www.lit.edu.cn/jwc/jwwlglxt.htm)。

    # 2.点击“学生成绩—重修报名”，在右侧窗口内的课程信息中，选择需要重修课程，点击“申请”即可。

    # 3.缴费

    学生缴费时间: 以学校通知为准，可咨询辅导员。 缴费方式:校园统一支付平台自助缴费 手机移动端登录，关注"洛阳理工学院财务处”微信公众号，点击“学生收费”- "学生缴费”进入洛阳理工学院校园统-支付平台登录界面。用户名为学号(请注意:2019-2020级学号带字母,如B20010101，其他年级学号不带字母)，初始密码为身份证后6位(未尾为字母时大写)。登录成功后,点击“学费缴费”，勾选自己要交的收费项目，(不支持修改缴费金额) ”点击右下角“缴费"按钮，选择支付方式(请选择农行聚合的支付方式，支持微信和各种银行卡)，确认支付后完成缴费。 注:如本人手机无法登录系统缴费，可电话咨询65928121。

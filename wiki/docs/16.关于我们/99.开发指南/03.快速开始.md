## 2. 本地开发 <Badge text="Dev"/>



<code-group>
  <code-block title="npm" active>

```bash
# clone the project
git clone https://github.com/$username$/homepage.git
# enter the project directory
cd homepage
# install dependency
npm install 
# develop
npm run dev 
```
  </code-block>

  <code-block title="yarn">
```bash
# clone the project
git clone https://github.com/$username$/homepage.git
# enter the project directory
cd homepage
# install dependency
yarn install
# develop
yarn dev
```
  </code-block>
</code-group>
